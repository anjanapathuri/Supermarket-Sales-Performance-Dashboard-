🛒 Supermarket Sales Performance Dashboard | Power BI

 📊 Overview
This project is an interactive Power BI dashboard designed to analyze supermarket sales performance across products, regions, and time periods. It provides key business insights through KPIs, trends, and visual analytics to support data-driven decision-making.

 🎯 Business Objective
To analyze supermarket sales data, monitor business performance, identify sales trends, and support better operational and strategic decision-making.

 🚀 What I Did
- Cleaned and prepared the dataset using Excel  
- Created DAX measures for KPIs like Sales, Profit, Orders, and Return Rate  
- Designed interactive visuals and dashboard layout in Power BI  
- Built a complete dashboard to analyze overall business performance  

  📊 Key Features
- KPI Metrics (Total Sales, Profit, Orders, Return Rate)  
- Product-wise Performance Analysis  
- City/Region-wise Sales using Map Visualization  
- Monthly and Daily Sales Trend Analysis  
- Revenue vs Target Comparison  
- Interactive Filters and Slicers for dynamic insights  

 🛠️ Tools Used
- Power BI  
- Microsoft Excel (Data Preparation)  

 📌 Key Insights
- Identified top-performing products contributing the highest revenue  
- Observed clear sales trends across different time periods  
- Regional analysis helped identify high and low-performing areas  
- Low return rate indicates strong product quality and customer satisfaction  
- Enabled better business decisions through clear and interactive insights  

 💡 Business Impact
This dashboard helps business teams monitor sales performance, identify growth opportunities, and optimize product, customer, and regional strategies for improved profitability.
